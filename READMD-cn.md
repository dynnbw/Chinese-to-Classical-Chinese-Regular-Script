# 古籍篆楷转换台

一个将繁体中文转换为篆书楷化字的在线工具，让您的“对手”体验古文字之美。

## 项目简介

本项目是一个基于Web的篆书楷化字转换工具，可以将繁体中文转换为篆书楷化字。工具支持实时转换、字体自定义、数据导入导出等功能，专为“正体字”爱好者设计。

## 重要说明：关于字形与读音

**请务必注意**：本工具转换的篆书楷化字并不一定对应现代汉字的标准读音。转换是基于字形结构，而非字音。

###字形来源与转换原则
本工具采用的篆书字形主要来源于以下古代字书与文献：
- 《说文解字》
- 《说文解字注》
- 《康熙字典》
- 《正字通》

**转换原则如下**：
1. **主从原则**：优先采用《说文解字》中收录的小篆字形提取字形。
2. **次从原则**：若《说文解字》中无对应字形，则采用其附录的“古文”（通常指大篆或更古的字形）提取字形。
3. **辅助原则**：以上皆无时，参考其他可信古籍中的字形。
4. **从古原则**：当不同现代汉字拥有相同部首时，该部首的转换尽量统一采用其古体写法。
5. **从新原则**：若某个古字形在当前的Unicode 标准中没有对应字符，则选用一个字形最为接近的现有汉字作为其楷化字。

**示例**：
- “𤁉”与“仅”拥有相同部首“堇”，因此该部首的转换理应统一采用古体字形，由于由于并没有对应部首的字符则采用最为接近的现有汉字。

### 因此，工具输出的文字主要供字形研究、参考与艺术设计之用，**请勿将其直接等同于现代汉字的标准写法或读音**。

## 功能特点

### 核心功能
- **篆书楷化转换**：将繁体字转换为对应的篆书楷化字
- **实时转换**：支持输入时自动实时转换
- **简繁转换**：集成第三方简繁转换工具
- **字符码点显示**：可选显示Unicode码点信息

### 字体支持
- **云端字体**：预置多款支持扩展汉字的字体
- **系统字体**：使用系统默认字体
- **自定义字体**：支持通过URL加载自定义字体

### 数据管理
- **映像表导入**：支持从URL导入JSON格式的映像数据
- **结果导出**：将转换结果导出为文本文件

### 统计信息
-显示映像总数、成功转换次数等统计信息
-转换统计

## 快速开始

### 在线使用
直接访问[此项目Github Pages](https://dynnbw.github.io/Chinese-to-Classical-Chinese-Regular-Script/)即可使用。

# 使用说明

## 界面设计与显示差异

## 移动端界面（宽度≤768px）
###布局调整
- **横排模式**：自动切换为传统的从左到右横排布局

- **弹性布局**：控制面板调整为横向排列，按钮水平展示

- **触控优化**：按钮尺寸增大，便于手指操作

- **面板全屏**：字体设置和简繁转换面板占据屏幕大部分区域

### 交互特点
- 直排文字区域转换为横排输入

- 按钮文字方向调整为水平

- 工具提示位置调整为按钮下方

- 比较结果显示为水平排列（原文→转换结果）

- 尺寸优化
- 主标题字体缩小至1.5em

- 输入区域高度固定为8em

- 控制面板按钮紧凑排列

- 状态栏信息简化显示

## 桌面端界面（宽度> 768px）
### 传统风格
- **直排模式**：从右到左的垂直书写方式

- **古籍布局**：模拟古籍书卷的阅读体验

- **精细控制**：按钮和工具提示使用直排文字

### 专业界面设计
- 控制面板固定在右侧，宽度9em

- 主内容区域占据大部分空间

- 字体设置面板位于屏幕中央，尺寸适中

- 简繁转换工具以iframe形式嵌入

- 输入区域宽度调整为12em

- 面板位置微调至更适合触控

- 字体大小和间距适当缩小

- 布局流式变化：根据屏幕宽度自动调整布局

- 字体大小缩放：确保在不同设备上的可读性

- 触控/鼠标优化：针对不同输入方式优化交互

- 面板位置智慧调整：避免遮挡主要内容

### 基本转换
1. 在左侧输入框输入繁体中文
2. 点击「执行转换」按钮或启用实时转换
3. 在右侧查看转换结果

### 字体设置
1. 点击控制面板的「字体设置」按钮
2. 选择云端、系统字体
3. 预览并应用选择的字体

### 数据导入
1. 点击「导入数据」按钮
2. 输入包含映像数据的JSON文件URL
3. 系统将自动加载并合并映像表

## 字符映像原理

项目使用预定义的映像表将现代汉字转换为篆书楷化字：

```javascript
{
“天”：“𠀘”，
“地”：“𡍑”，
“日”：“𡆠”，
“月”：“𡴑”
}
```
