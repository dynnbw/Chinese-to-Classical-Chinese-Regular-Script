# 古籍篆楷轉換臺

一個將繁體中文轉換為篆書楷化字的在線工具，讓您的"對手"體驗古文字之美。

## 項目簡介

本項目是一個基於Web的篆書楷化字轉換工具，可以將繁體中文轉換為篆書楷化字。工具支持實時轉換、字體自定義、數據導入導出等功能，專為"正體字"愛好者設計。

## 重要說明：關於字形與讀音

**請務必注意**：本工具轉換的篆書楷化字並不一定對應現代漢字的標準讀音。轉換是基於字形結構，而非字音。

### 字形來源與轉換原則
本工具採用的篆書字形主要來源於以下古代字書與文獻：
- 《說文解字》
- 《說文解字注》
- 《康熙字典》
- 《正字通》

**轉換原則如下**：
1.  **主從原則**：優先採用《說文解字》中收錄的小篆字形選取字形。
2.  **次從原則**：若《說文解字》中無對應字形，則採用其附錄的“古文”（通常指大篆或更古的字形）選取字形。
3.  **輔助原則**：以上皆無時，參考其他可信古籍中的字形。
4.  **從古原則**：當不同現代漢字擁有相同部首時，該部首的轉換盡量統一採用其古體寫法。
5.  **從新原則**：若某個古字形在當前的 Unicode 標準中沒有對應字符，則選用一個字形最為接近的現有漢字作為其楷化字。

**示例**：
- “𤁉”與“僅”擁有相同部首“堇”，因此該部首的轉換理應統一採用古體字形，由于由於並沒有對應部首的字符則採用最為接近的現有漢字。

### 因此，工具輸出的文字主要供字形研究、參考與藝術設計之用，**請勿將其直接等同於現代漢字的標準寫法或讀音**。

## 功能特點

### 核心功能
- **篆書楷化轉換**：將繁體字轉換為對應的篆書楷化字
- **實時轉換**：支持輸入時自動實時轉換
- **簡繁轉換**：集成第三方簡繁轉換工具
- **字符碼點顯示**：可選顯示Unicode碼點信息

### 字體支持
- **雲端字體**：預置多款支持擴展漢字的字體
- **系統字體**：使用系統默認字體
- **自定義字體**：支持通過URL加載自定義字體

### 數據管理
- **映射表導入**：支持從URL導入JSON格式的映射數據
- **結果導出**：將轉換結果導出為文本文件

### 統計信息
- 顯示映射總數、成功轉換次數等統計信息
- 轉換統計

## 快速開始

### 在線使用
直接訪問[此項目Github Pages](https://dynnbw.github.io/Chinese-to-Classical-Chinese-Regular-Script/)即可使用。

# 使用說明

## 界面設計與顯示差異

## 移動端界面（寬度 ≤ 768px）
### 布局調整
- **橫排模式**：自動切換為傳統的從左到右橫排佈局

- **彈性佈局**：控制面板調整為橫向排列，按鈕水平展示

- **觸控優化**：按鈕尺寸增大，便於手指操作

- **面板全屏**：字體設置和簡繁轉換面板佔據螢幕大部分區域

### 交互特點
- 直排文字區域轉換為橫排輸入

- 按鈕文字方向調整為水平

- 工具提示位置調整為按鈕下方

- 比較結果顯示為水平排列（原文 → 轉換結果）

- 尺寸優化
- 主標題字體縮小至1.5em

- 輸入區域高度固定為8em

- 控制面板按鈕緊湊排列

- 狀態欄信息簡化顯示

## 桌面端界面（寬度 > 768px）
### 傳統風格
- **直排模式**：從右到左的垂直書寫方式

- **古籍佈局**：模擬古籍書卷的閱讀體驗

- **精細控制**：按鈕和工具提示使用直排文字

### 專業界面設計
- 控制面板固定在右側，寬度9em

- 主內容區域佔據大部分空間

- 字體設置面板位於螢幕中央，尺寸適中

- 簡繁轉換工具以iframe形式嵌入

- 輸入區域寬度調整為12em

- 面板位置微調至更適合觸控

- 字體大小和間距適當縮小

- 布局流式變化：根據螢幕寬度自動調整布局

- 字體大小縮放：確保在不同設備上的可讀性

- 觸控/滑鼠優化：針對不同輸入方式優化交互

- 面板位置智慧調整：避免遮擋主要內容

### 基本轉換
1. 在左側輸入框輸入繁體中文
2. 點擊「執行轉換」按鈕或啟用實時轉換
3. 在右側查看轉換結果

### 字體設置
1. 點擊控制面板的「字體設置」按鈕
2. 選擇雲端、系統字體
3. 預覽並應用選擇的字體

### 數據導入
1. 點擊「導入數據」按鈕
2. 輸入包含映射數據的JSON文件URL
3. 系統將自動加載並合並映射表

## 字符映射原理

項目使用預定義的映射表將現代漢字轉換為篆書楷化字
- 示例：
```javascript
{
"天": "𠀘",
"地": "𡍑",
"日": "𡆠",
"月": "𡴑"
}
```
