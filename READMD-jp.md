# 篆書楷書変換プラットフォーム

繁体字を篆書楷化文字に変換するオンラインツール。あなたの「相手」に古代文字の美しさを体験させましょう。

## プロジェクト概要

このプロジェクトは、Webベースの篆書楷化文字変換ツールです。リアルタイム変換、フォントカスタマイズ、データ入出力等功能をサポートし、「正字体」愛好家のために設計されています。

## 重要な説明：字形と発音について

**必ずお読みください**：本ツールが変換する篆書楷化文字は、現代中国語の標準的な発音に対応するとは限りません。変換は字音ではなく、字形構造に基づいて行われます。

### 字形の出典と変換原則

本ツールで使用する篆書字形は、主に以下の古代字書と文献に由来します：
- 『説文解字』
- 『説文解字注』
- 『康熙字典』
- 『正字通』

**変換原則は以下の通りです**：
1.  **主原則**：『説文解字』に収録されている小篆字形を優先的に採用します。
2.  **副原則**：『説文解字』に対応字形がない場合は、その付録の「古文」（通常は大篆またはより古い字形を指す）から字形を採用します。
3.  **補助原則**：上記のいずれにもない場合は、他の信頼できる古籍の字形を参考にします。
4.  **従古原則**：異なる現代漢字が同じ部首を持つ場合、その部首の変換は可能な限りその古体で統一します。
5.  **従新原則**：ある古字形が現在のUnicode標準に対応する文字を持たない場合、字形が最も近い既存の漢字をその楷化字として選びます。

**例**：
- 「𤁉」と「僅」は同じ部首「堇」を持つため、この部首の変換は原則として古体字形を統一して採用すべきです。対応する部首の文字がないため、最も近い既存の漢字が採用されます。

### したがって、本ツールが出力する文字は、主として字形の研究、参照、芸術デザインに供するものです。**現代漢字の標準的な書体や発音と直接同一視しないでください。**

## 機能の特徴

### コア機能
- **篆書楷化変換**：繁体字を対応する篆書楷化文字に変換します。
- **リアルタイム変換**：入力中の自動リアルタイム変換をサポートします。
- **簡体字繁体字変換**：サードパーティの簡体字繁体字変換ツールを統合しています。
- **文字コードポイント表示**：Unicodeコードポイント情報の表示オプションがあります。

### フォントサポート
- **クラウドフォント**：拡張漢字をサポートする複数のフォントがプリインストールされています。
- **システムフォント**：システムデフォルトのフォントを使用します。
- **カスタムフォント**：URLを通じてカスタムフォントを読み込むことをサポートします。

### データ管理
- **マッピングテーブルインポート**：URLからJSON形式のマッピングデータをインポートすることをサポートします。
- **結果エクスポート**：変換結果をテキストファイルとしてエクスポートします。

### 統計情報
- マッピング総数、変換成功回数などの統計情報を表示します。
- 変換統計

# 使用説明

## インターフェース設計と表示の違い

## モバイル端末インターフェース（幅 ≤ 768px）
### レイアウト調整
- **横並びモード**：自動的に左から右への横書きレイアウトに切り替わります。

- **フレキシブルレイアウト**：コントロールパネルを横並びに調整し、ボタンを水平に表示します。

- **タッチ操作最適化**：ボタンサイズを大きくし、指での操作をしやすくします。

- **パネル全画面表示**：フォント設定と簡体字繁体字変換パネルが画面の大部分を占めます。

### インタラクションの特徴
- 縦書きテキスト領域が横書き入力に変換されます。

- ボタンの文字方向が水平に調整されます。

- ツールチップの位置がボタンの下に調整されます。

- 比較結果は水平に表示されます（原文 → 変換結果）。

- サイズ最適化
- メインタイトルのフォントサイズを1.5emに縮小します。

- 入力領域の高さを8emに固定します。

- コントロールパネルのボタンをコンパクトに配置します。

- ステータスバーの情報表示を簡素化します。

## デスクトップ端末インターフェース（幅 > 768px）
### 伝統的スタイル
- **縦書きモード**：右から左への垂直書き方式です。

- **古典的レイアウト**：古典書籍の読書体験を模倣します。

- **精密制御**：ボタンとツールチップは縦書き文字を使用します。

### プロフェッショナルインターフェース設計
- コントロールパネルは右側に固定、幅9em。

- メインコンテンツ領域が大部分のスペースを占めます。

- フォント設定パネルは画面中央に、適切なサイズで表示されます。

- 簡体字繁体字変換ツールはiframe形式で埋め込まれます。

- 入力領域の幅を12emに調整します。

- タッチ操作に適するようパネル位置を微調整します。

- フォントサイズと間隔を適切に縮小します。

- 流動的レイアウト変化：画面幅に基づいて自動的にレイアウトを調整します。

- フォントサイズスケーリング：異なる端末での可読性を確保します。

- タッチ/マウス最適化：異なる入力方法に合わせてインタラクションを最適化します。

- パネル位置の智能的調整：主要コンテンツの遮蔽を避けます。

### 基本変換
1. 左側の入力ボックスに繁体字中国語を入力します。
2. 「変換実行」ボタンをクリックするか、リアルタイム変換を有効にします。
3. 右側で変換結果を確認します。

### フォント設定
1. コントロールパネルの「フォント設定」ボタンをクリックします。
2. クラウドフォント、システムフォントから選択します。
3. 選択したフォントをプレビューして適用します。

### データインポート
1. 「データインポート」ボタンをクリックします。
2. マッピングデータを含むJSONファイルのURLを入力します。
3. システムが自動的にマッピングテーブルを読み込み、統合します。

## 文字マッピング原理

プロジェクトは事前定義されたマッピングテーブルを使用して、現代漢字を篆書楷化文字に変換します。
- 例：
```javascript
{
"天": "𠀘",
"地": "𡍑",
"日": "𡆠",
"月": "𡴑"
}
```
